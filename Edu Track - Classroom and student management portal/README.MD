The EduTrack Portal is a full-stack Java-based web + client application designed to manage and streamline academic activities. The portal provides CRUD operations for managing batches along with detailed student records in each batch. It leverages Spring Boot for building a RESTful backend, MongoDB for scalable data
storage, and a Java client application for user interaction. The system simplifies batch creation, student management, and record updates, making it a centralized classroom management solution.
